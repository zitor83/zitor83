- 🖥️ **Deployed on a VPS** (Ubuntu) using **Docker** containers orchestrated behind an **Nginx** reverse proxy

### 🗓️ Event Management API

> A production-ready REST API demonstrating enterprise-grade security and architecture patterns.

- 🔑 **Stateless security** with **JWT** (via **JJWT**) — zero session state, full scalability
- 🛡️ **RBAC (Role-Based Access Control)** enforced with Spring Security's `@PreAuthorize`
- 🔄 **Advanced DTO mapping** with **MapStruct** for clean, maintainable data transformations
- 🏛️ Built on **Hexagonal Architecture** separating domain logic from infrastructure concerns
